# Expense Tax Mapping Guidelines

Deductibility codes map as follows:

1. **Meals & Entertainment**: Map to `ME-50` (50% corporate tax deduction).
2. **Travel & Lodging**: Map to `TRV-100` (100% deduction).
3. **Office Supplies**: Map to `OFF-100` (100% deduction).
4. **Software / SaaS**: Map to `SaaS-100` (100% deduction).
5. **Other / Uncategorized**: Map to `GEN-TAX` (standard review mapping).
