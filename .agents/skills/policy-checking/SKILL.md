# Expense Policy Checking Guidelines

Detailed checks to run for each corporate claim:

1. **Daily Meals limit**: Flag meals exceeding $75.00.
2. **Software subscriptions**: Flag purchases exceeding $500.00.
3. **Weekend Transactions**: Highlight any claim dated Saturday or Sunday.
4. **Receipt Requirements**: Flags if amount is > $25.00 and no receipt is attached.
