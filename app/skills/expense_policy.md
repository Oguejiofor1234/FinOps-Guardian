# Corporate Expense Policy Rules

This policy document contains guidelines for evaluating corporate expenses.

## 1. Core Expense Limits and Rules
* **Daily Meals limit:** $75.00 per person per day.
* **Travel Booking:** All flights and hotels must be booked at least 14 days in advance. First-class travel is strictly prohibited without executive sign-off.
* **Receipt Requirement:** Any transaction greater than $25.00 MUST have a valid receipt.
* **Software Purchases:** Subscriptions/licenses must not exceed $500.00 without IT approval.
* **Weekend Claims:** Expenses incurred on weekends (Saturday & Sunday) are flagged for manager audit unless travel itinerary is attached.

## 2. Tax Code Mapping Guidelines
* **Meals & Entertainment:** Map to tax code `ME-50` (50% deductible).
* **Office Supplies:** Map to tax code `OFF-100` (100% deductible).
* **Software/SaaS:** Map to tax code `SaaS-100` (100% deductible).
* **Travel & Lodging:** Map to tax code `TRV-100` (100% deductible).
* **Uncategorized/Other:** Map to tax code `GEN-TAX` (General tax/review required).
